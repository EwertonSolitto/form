# Installation 🔧

Make sure to have **NodeJS** and **Expo** installed.

```bash

# Clone repository
git clone https://github.com/EwertonSolitto/login-app.git

# Go to the directory
cd login-app

# Install dependencies
npm i

# Start Application
npm start

```
